Struct Tile

A wrapper around Object to give some clarity to the code. Tiles have no semantics except Equals and GetHashCode - all other behaviour is specified externally when configuring other DeBroglie classes. Arrays of Tile are the main objects that are generated by the DeBroglie librarie, and the Tile struct is used elsewhere to indicate the relationship to generation.
